Fandom Guess is an offline app-game for groups of two or more people.

The main goal of the game is to get your team (or partner if playing in pairs) to guess a specific fandom-related word or phrase without using a list of forbidden terms provided on the card. The team earns points for each correctly guessed word.

Players need to be resourceful and eloquent to bypass the listed restrictions. The game requires quick thinking, creativity, and a good knowledge of the fandom, making it incredibly engaging and fun for all participants.

# Installation
1. Download the .apk file
2. Transfer the .apk file to your phone (through USB port if necessary)
3. Open Internal storage on your phone and open fandom_guess.apk with package  installer
4. You may get the security warning. Go to your settings and allow permission to install unknown apps
5. Click install
6. Now you can play with your friends!
